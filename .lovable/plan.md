
# Portfolio-Projekte aus PDFs extrahieren und erstellen

## Uebersicht

Alle 13 Projekte werden aus den beiden PDF-Dateien extrahiert und auf der Startseite sowie der Projects-Seite angezeigt. Die Bilder werden kopiert und die Projektdaten in `projects.ts` eingefuegt.

---

## PDF-Seitenstruktur

### PDF Teil 1: Seiten 1-50 (S.11-50 = Page 11-50 im PDF)
- PDF-Seite 11 = img_p10_* (Seitenzahl - 1)
- Beispiel: S.11 entspricht img_p10, S.17 entspricht img_p16

### PDF Teil 2: Seiten 51-98 (S.51 = Page 1 im PDF)
- PDF-Seite 1 = S.51 = img_p0_*
- Beispiel: S.61 entspricht img_p10, S.67 entspricht img_p16

## Technische Umsetzung

### Schritt 1: Projektordner erstellen und Bilder kopieren

Fuer jedes Projekt wird ein Ordner in `public/images/` erstellt:
- public/images/infrarotthermometer/
- public/images/l-ement/
- public/images/oru/
- public/images/glasbaustein/
- public/images/atobe/
- public/images/glug/
- public/images/glas-metall-fliese/
- public/images/grow-mushrooms/
- public/images/fast-good-cheap/
- public/images/blender-workshop/
- public/images/3d-print-experiments/
- public/images/workshops/
- public/images/sonderanfertigungen/

### Schritt 2: Bilder von parsed-documents kopieren

Die Bilder werden aus den parsed-documents Verzeichnissen kopiert:
- PDF Teil 1: `parsed-documents://20260201-142511-201473/Clim_Michel_Portfolio_2025_S1-50-2.pdf/images/`
- PDF Teil 2: `parsed-documents://20260201-142523-093312/Clim_Michel_Portfolio_2025_S51-98-2.pdf/images/`

**Filtering:**
- Nur .jpg Dateien werden standardmaessig verwendet
- .png Dateien werden geprueft (oft Renderings/Grafiken)
- Seiten 49-50 von glug! werden ausgeschlossen
- Erstes Bild von Glas-Metall-Fliese (S.61) wird ausgeschlossen

### Schritt 3: projects.ts aktualisieren

Die Datei `src/data/projects.ts` wird mit allen 13 Projekten gefuellt:

```text
export const projects: Project[] = [
  {
    id: 'infrarotthermometer',
    title: 'Infrarotthermometer',
    description: 'Das Fieberthermometer ist ein essenzielles Produkt...',
    category: 'Product Design',
    materials: ['Bio Kunststoff', 'Li-Ion-Akku', 'SMD Elektronik'],
    techniques: ['Clay-Modellierung', 'Ergonomie', '3D Scanning'],
    previewImages: ['/images/infrarotthermometer/1.jpg', ...], // max 3
    images: ['/images/infrarotthermometer/1.jpg', ...], // alle
    year: '2019',
    month: 'June',
    displayFormat: 'portrait'
  },
  // ... weitere 12 Projekte
];
```
